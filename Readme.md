#  Real-Time Chat Application

A real-time chat application built using **HTML, CSS, JavaScript, Node.js, and WebSocket**. This project allows multiple users to communicate instantly without refreshing the page.

---


---

##  Features

-  Real-time messaging
-  Instant message delivery using WebSocket
-  Multiple users can chat simultaneously
-  Responsive user interface
-  Clean and modern design
-  Automatic message updates
-  Fast and lightweight

##  Installation

### 1. Clone the Repository

```bash
git 
```

### 2. Open the Project

```bash
cd real-time-chat-app
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Server

```bash
node server.js
```

### 5. Open in Browser

```
http://localhost:3000
```

---

##  How It Works

1. The server starts using Node.js.
2. Clients connect through WebSocket.
3. Messages are sent instantly to the server.
4. The server broadcasts messages to all connected users.
5. Every connected user receives the message in real time.
